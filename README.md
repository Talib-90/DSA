# DSA Practice in Python

This repository contains my solutions to common **Data Structures and Algorithms (DSA)** problems, implemented in Python. The goal is to build strong problem-solving skills through consistent practice and clear implementation.

---

## Overview

The repository includes solutions to fundamental DSA problems such as:

* Searching (e.g., Binary Search)
* Sorting (e.g., Bubble Sort, Selection Sort, Insertion Sort)
* Arrays and Numbers (e.g., Two Number Sum, Product Sum)
* Recursion and Tree Problems (e.g., Node Depths, Branch Sum)
* String Manipulation (e.g., Palindrome Check, Caesar Cipher)

Each file focuses on a single problem and its solution.

---

## Structure

* Each `.py` file represents one problem
* File names reflect the problem being solved
* Code is written for clarity and learning, not optimization alone

Example:

```
BinarySearch.py
BubbleSort.py
NodeDepths.py
TwoNumberSum.py
```
